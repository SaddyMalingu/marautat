Tenant Onboarding

  1. Welcome & Introduction

    Welcome to the Alphadome Tenant Onboarding! This step-by-step guide will help you set up your business and WhatsApp bot. Estimated time: 10-20 minutes.

  2. Business Information

    Provide your business name, contact details, and admin info in the portal.
    Why? This helps us personalize your experience and support.

  3. WhatsApp Business API Setup

    • Register a WhatsApp Business Account (WABA).
    • Create a Facebook Business Manager account (if you don’t have one).
    • Complete Facebook Business Verification (required for WhatsApp API access).
    • Go to Business Settings → WhatsApp Accounts to add your phone number and set up your WhatsApp profile.
    • Generate your WhatsApp API credentials:
      • App ID & App Secret: Facebook Developers → My Apps → Select your app → Settings > Basic.
      • Phone Number ID & WABA ID: Business Settings → WhatsApp Accounts → Select your account → Settings.
      • Access Token: Cloud API Get Started → Generate Token.
    Checklist:
    • App ID
    • App Secret
    • Phone Number ID
    • WABA ID
    • Access Token
    Need help? See the official WhatsApp Cloud API guide or contact support for step-by-step assistance.

  4. Bot Configuration

    Enter your WhatsApp API credentials in the Bot Config section of the portal. Use the Test Connection button to verify integration. If you see a green check, you’re ready!

  5. Feature Enablement

    Choose which features to enable for your business: Catalog, Campaigns, Dashboard, and more. You can enable/disable features anytime in the portal.

  6. Media & Catalog Upload

    Upload your product catalog and media files. Download CSV Template for easy import. Follow the instructions in the portal for supported formats.

  7. Testing & Go-Live

    Review your setup, send a test message, and confirm you’re ready to go live. Use the Test Message button in the portal to verify everything works as expected.

  8. Support & Resources

    Need help? Contact Support or view the Setup Guide.
    More resources: WhatsApp Cloud API Docs, WABA Getting Started
